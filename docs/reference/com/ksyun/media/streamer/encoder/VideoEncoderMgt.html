Class Overview

Video encoder manager for convenient usage. This module had two input pin, one accept ImgTexFrame and another accept ImgBufFrame, which to use determined by which encode method in use.
METHOD_HARDWARE and METHOD_SOFTWARE would use ImgTexFrame as input frame, besides, METHOD_SOFTWARE_COMPAT would use ImgBufFrame.
