Class Overview

硬编即在使用金山云推流SDK时使用Android硬件编码 硬编默认需要在4.3 (18)以上系统 金山云服务器根据实际测试结果存储了一部分机型系统的硬编支持情况 可以通过该类从服务端获取当前机型是否在硬编白名单中，并且把相关信息存储在本地 建议在APK联网后调用init接口 通过getDeviceInfo可以获取到机型信息，默认硬编为不支持硬编
